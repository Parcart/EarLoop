# Анализ важности частот эквалайзера (SocialFX)

## Общее описание

В этом проекте проводится анализ **важности частот эквалайзера**
на основе датасета **SocialFX** с целью:

- понять, какие частотные области чаще и сильнее всего корректируются
- оценить согласованность эквалайзерных решений
- выделить набор частот, пригодный для сокращённого и интерпретируемого EQ
- подготовить основу для сравнения с AutoEQ

Анализ основан **только на параметрах эквалайзера** и не использует аудиосигнал.

---

## Датасет

### SocialFX

Используется параметрическая часть датасета SocialFX.
Каждая строка описывает одну полосу эквалайзера и содержит:

- `fc_hz` — центральная частота (Гц)
- `gain_db` — усиление / ослабление (дБ)
- `ratings_consistency` — степень согласованности (доверия) между оценками

Данные приведены к *long-формату* (`eq_long_m`),
где каждая строка соответствует одной настройке полосы.
