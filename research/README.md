
## Описание этапов

### 1. AutoEQ

Папка `autoeq/` содержит анализ эталонных кривых AutoEQ,
которые характеризуются более агрессивной и структурированной эквализацией.
Данный датасет используется для выявления характерных частотных паттернов
и как источник дополнительной структурной информации.

### 2. SocialFX

Папка `socialfx/` содержит анализ датасета SocialFX,
представляющего реальные пользовательские решения эквализации.
В рамках анализа:
- вычисляются статистики усиления и ослабления (boost / cut);
- анализируются распределения усилений по частотам;
- формируется статистическая основа для последующего моделирования.

### 3. SocialFX & AutoEQ

Папка `socialfx&autoeq/` посвящена совместному анализу двух датасетов.
В рамках данного этапа:
- данные приводятся к общей логарифмической сетке частот;
- сравниваются характеристики пользовательской и эталонной эквализации;
- формируется объединённая кривая важности частот,
используемая для проектирования архитектуры компактного эквалайзера.

### 4. Compact EQ Design

Папка `compact_eq_design/` содержит основной этап проектирования
компактного параметрического эквалайзера.
В рамках данного этапа:
- выполняется сокращение числа полос эквалайзера;
- проводится sweep по числу полос;
- используется knee-point анализ для выбора оптимальной сложности модели;
- выполняется количественное сравнение компактного эквалайзера
  с 40-полосным пользовательским эквалайзером SocialFX.

## Связь с основной моделью

Данная папка содержит исключительно исследовательские материалы
и не включает реализацию конечной модели или интерфейса.
