# Проектирование компактного параметрического эквалайзера

## Назначение

В данной папке выполняется проектирование компактной архитектуры
параметрического эквалайзера,
аппроксимирующей статистику пользовательских решений SocialFX.

Целью является сокращение числа полос эквалайзера
при минимальной потере информации,
с последующим использованием данной архитектуры
в персонализированной модели эквализации.

## Содержимое

- `compact_eq_design.ipynb` — основной ноутбук проектирования эквалайзера;

## Основные этапы

В рамках данного этапа:
- выполняется sweep по числу полос эквалайзера;
- проводится knee-point анализ зависимости ошибки от сложности модели;
- выбирается оптимальное число полос;
- анализируется компромисс между точностью и компактностью;
- формируется финальная архитектура эквалайзера.

Форма фильтров (Q-параметры) и сглаживание
рассматриваются как отдельный этап развития модели
и в данной работе не оптимизируются.
